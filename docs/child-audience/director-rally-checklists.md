# Age-band director rally checklists

> **Stub** — per–age-band checklists for programming directors: what to **rally into your zone** (infrastructure, owned assets, booked activators, schedule grammar). Each activation should earn **cohort credits** when zone, time window, and [fit tags](fit-tags.html) align — not a single “family-friendly” label for the whole site.

See also: [Site plan & programming](site-and-zoning.html) · [Cohorts C0–C3](cohorts-young.html) · [Cohort C4 — teens](cohorts-teens.html) · [Implementation](implementation.html)

---

## How to use

1. **Pick your band** (director lane) below.
2. Work top to bottom — **infrastructure before programming**.
3. Log each booked activation on the run-of-show with: zone · time · fit tags · competition layer (focal / async / none).

**Cert row (one line per activation):** Activation · Type (owned / booked) · Zone · Time window · Fit tags · Competition layer

## C0 — Stroller & carrier (≈0–2)

**Director lane:** C0 lead  
**Rally zone:** Village core — shade path, changing, quiet rest, stroller parking in sight of corral  
**Veto window:** First 20 minutes after village entry (arrival-relative, not clock time)

### Space & infrastructure

- [ ] Diaper changing — not in food zone
- [ ] Continuous shade path from gate to village
- [ ] Quiet rest / nursing corral
- [ ] Stroller paths min ~36–42 in audited
- [ ] Feeding + water refill point
- [ ] Sound discipline near rest zone (no surprise loud acts adjacent)
- [ ] Low-height décor at stroller sightline (24–36 in)

### Owned assets to deploy

- [ ] Low-height décor kit *(coalition — TBD SKU)*
- [ ] Stroller-mount lantern / LED pilot *(TBD — procession handoff)*

### Booked activators to rally

- [ ] Anchor host — parent–infant movement (Zumbini-class)
- [ ] Sensory touch table (low volume)
- [ ] Library / storytime partner — pre-event ritual optional
- [ ] Touch-a-truck open house — heritage hero hook, flat viewing

### Programming to schedule

- [ ] Shade path + slow décor — always-on, no queue
- [ ] Roving music — low volume only
- [ ] Themed stroller parade pre-procession (optional)
- [ ] **Skip:** main-stage front row · carnival mid-day lines · queue-only activations

### Publish & measure

- [ ] Fit tags: `stroller-ok` · `shade` · `quiet-exit`
- [ ] Discovery listing with age band in title (e.g. *“0–2 welcome — changing @ Village”*)
- [ ] Success: one shade loop without stroller reverse · calm dwell in corral

### Do not mix into C0 peak

- [ ] Bounce houses without adjacent shade
- [ ] Skill challenge / Challenge row
- [ ] DIY game stations as headline product

## C1 — Walking toddler (≈2–4)

**Director lane:** C1 lead  
**Rally zone:** Bounded village corral · theater pocket with obvious exit  
**Veto window:** First 20 minutes — character loop or easy win required

### Space & infrastructure

- [ ] Soft edges · exit visible from every station
- [ ] Duplicate stations to shorten lines
- [ ] Shade adjacent to any active play station

### Owned assets to deploy

- [ ] Children's Theater pocket kit *(TBD — soft backdrop, low platform, shade)*

### Booked activators to rally

- [ ] Roving characters — ~every 20 min until early afternoon
- [ ] Children's Theater activator — 15–20 min sets
- [ ] Bubble mat / parallel-play stations
- [ ] Story circle host (bilingual if possible)
- [ ] Local character performers — heritage skin, not licensed IP
- [ ] Touch-a-truck — bounded open house

### Programming to schedule

- [ ] Make-and-walk (craft → short loop) — 10 min blocks
- [ ] Photo corral / meet backdrop
- [ ] **Skip:** seated 30-min shows · competitive games

### Publish & measure

- [ ] Fit tags: `parallel-play` · `roving-character` · `exit-visible`
- [ ] Success: repeat station visits · no meltdown at zone edge

### Do not mix into C1 peak

- [ ] Open bounce without shade next door *(bounce houses = separate registry + insurance sign-off)*
- [ ] Challenge row / tween skill heats
- [ ] Undifferentiated DIY game-station drop-in

## C2 — Preschool (≈4–6)

**Director lane:** C2 lead (often paired with C1 for village)  
**Rally zone:** Village + craft row + family stage (15–20 min blocks)  
**Bridge:** Craft today → Sunday procession prop

### Space & infrastructure

- [ ] Craft row adjacent to village — handoff to take-home prop
- [ ] Family stage sightlines · early-exit path

### Owned assets to deploy

- [ ] Children's Theater pocket kit *(shared with C1)*
- [ ] Low-height décor *(shared with C0–C1)*

### Booked activators to rally

- [ ] Instrument petting zoo — 15 min structured block
- [ ] Stage magic / puppet — 15–20 min
- [ ] Children's Theater — heritage themes (Great Race, moon rabbit, zodiac; no licensed IP)
- [ ] Hands-on craft — lantern panel, zodiac mask
- [ ] Yard games rental *(optional)* — **junior time window only**; DIY drop-in; needs focal or async layer for school-age credit

### Programming to schedule

- [ ] Character loop or bubble mat on arrival
- [ ] Gateway show @ 12:00 or 12:45
- [ ] Craft → take-home prop @ 12:30+
- [ ] Parade @ 1:00 — can watch most; point out Sunday relay
- [ ] Heritage table or second theater @ 3:00
- [ ] Carnival *(optional, separate zone)* @ 4:00 — parent chooses spend; not bundled with theater

### Publish & measure

- [ ] Fit tags: `hands-on` · `passive-viewing-15m`
- [ ] Sunday hook published: craft prop → procession
- [ ] Success: names one character · completes craft · asks to return

### Do not mix into C2 peak

- [ ] Challenge row tween heats (same queue as 9–11)
- [ ] Teen Plaza programming inside village

## C3 — School age (≈6–12)

**Director lane:** C2–C3 lead  
**Rally zone:** Challenge row (village edge) · club home base · craft row gallery · parade route  
**Tween slice (≈9–11):** Same row — **tween-primary vibe**; never *“little kids zone”* copy

### Space & infrastructure

- [ ] Challenge row at village edge — not inside toddler corral sightline
- [ ] Club home base pins @ 12:30 (PTA banner, find-your-tribe board)
- [ ] Waiver / tagged activities for skill challenges
- [ ] Meetup plan signage for carnival split @ 4:00+

### Owned assets to deploy

- [ ] **Coalition-owned inflatable carnival games** — 8-station skill bundle (ring toss, dartboard, axe throw, baseball-style; shared blowers) · **not bounce houses**
- [ ] Deploy labor budgeted: trailer · setup/teardown crew · cleaning · portable power + fuel

### Booked activators to rally

- [ ] Martial / dance / sports clubs — named demo @ Challenge row
- [ ] Instrument petting zoo *(secondary; OK for multigen pause)*
- [ ] Classroom contest / People's Choice gallery partner
- [ ] Theater activator — enactment + mild challenge callbacks
- [ ] Repeat coalition activators with `school-age` fit tags
- [ ] Yard games rental — only with **focal time** or **async leaderboard** layered; weak alone at low crowd density

### Programming to schedule

- [ ] Scavenger stamp card — always-on
- [ ] Gateway show @ 12:00 or 12:45
- [ ] Home base opens @ 12:30 · named club slot on public grid
- [ ] Parade team intro @ 1:00 — Sunday relay CTA
- [ ] **Challenge row @ 2:00–4:00** — owned skill stations + club demos + relay photos
- [ ] Heritage module @ 3:00 (optional multigen)
- [ ] **Competition layer** (pick one per block):
  - [ ] **Focal:** scheduled heat · MC gather (e.g. *“Ring toss heat @ 1:15”*)
  - [ ] **Async:** leaderboard · stamp card · People's Choice · Sunday signup from photo
- [ ] Junior volunteer shift ~3 PM (badge + corral helper) — tween slice

### Time segmentation (required)

- [ ] **Tween-primary afternoon** (≈12:30–4:00) **OR** separate **junior morning block** — **not both mixed at peak**
- [ ] Junior morning window: younger school-age (6–8) · different copy · subset of stations

### Publish & measure

- [ ] Fit tags: `school-age` · `hands-on` · `predictable-schedule`
- [ ] Pre-event: named sessions · club parade slot · Sunday relay time
- [ ] Success: pre-festival submission · club named in parade · Sunday intent registered

### Do not mix into C3 Challenge row peak

- [ ] Toddler corral · stroller parade through skill queue
- [ ] DIY game-station drop-in without focal/async *(amusement only — no challenge credit)*
- [ ] Bounce houses in same lane as skill-challenge inflatables

## C4 — Teens (≈13–17)

**Director lane:** C4 liaison  
**Rally zone:** **Teen Plaza** + food corridor — **separate map pin**; not Children's Village headline  
**Gate question:** *“Would I stay past dinner with my people?”*

### Space & infrastructure

- [ ] Tables for 4–6 — friend groups, not stroller corral layout
- [ ] Charging · open crew board
- [ ] Meet-back corral time published
- [ ] Physical separation from toddler corral

### Owned assets to deploy

- [ ] Night-market / plaza furniture kit *(ops)*
- [ ] **Not** Challenge-row inflatables as teen headline — skill product lives in C3 grammar

### Booked activators to rally

- [ ] Affiliated youth-org meet — time posted on website before event
- [ ] Near-peer facilitator (≈16–19) for crew calls
- [ ] Youth performance slot — opt-in, not perform-for-adults gaze
- [ ] Maker / career micro-workshop partner
- [ ] Game lounge facilitator (mahjong intro, etc.)
- [ ] Activators tagged `peer-social` · `evening-ok` · `food-forward`

### Programming to schedule

- [ ] Plaza density @ 2:00+ — avoid empty plaza at 5 PM
- [ ] Interest block @ 3:00 (photo wall, playlist vote)
- [ ] **Crew call** — 15–25 min scavenger team / food crawl / photo mission
- [ ] Affiliated meet — evening anchor on public grid
- [ ] Headliner / evening corridor @ 6:00–8:00 if funded
- [ ] Parade optional — peer on relay team only

### Publish & measure

- [ ] Fit tags: `evening-ok` · `food-forward` · `peer-social` · `crew-call-HHMM` · `meet-back-corral` · `no-parent-zone`
- [ ] Teen-forward share asset (Story-sized — not parent social graphic only)
- [ ] Exit survey: *“Did you see anyone your age?”*

### Do not mix into C4 zone

- [ ] Parent-shadowed craft tables
- [ ] Children's Village framing · “kids tent” energy
- [ ] Challenge row billed as teen hangout without peer density

## Competition modes

| Mode | Best for | Example |
|------|----------|---------|
| **Focal time** | C3 tween · club tribes | *“Challenge row opens 12:30 · heat @ 1:15”* |
| **Async** | C3 · older tweens | Leaderboard · People's Choice · stamp card |
| **None (DIY drop-in)** | C2 at best | Open game stations without schedule or tribe — flag **no challenge credit** |

**Design rule:** Challenge row is **one vibe** — tween-primary afternoon **or** a separate junior morning block — not a mixed-age free-for-all at peak hours.
